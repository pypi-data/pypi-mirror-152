## Install it

```shell
pip install git+https://github.com/ffreemt/ezbee
# poetry add git+https://github.com/ffreemt/ezbee
# git clone https://github.com/ffreemt/ezbee && cd ezbee
```

## Use it
```python
ezbee --help
# or python -m ezbee --help
```
